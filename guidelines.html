🎾 SJTK TÄVLINGSANVISNINGAR

Version 2016-04

Regeltext anpassad för tävlingar anordnade av SJTK.
Denna text ska ses som en sammanfattning och anpassning för SJTK av de viktigaste punkterna ur "Tävlingshandboken" utgiven av Svenska Tennisförbundet.

📜 GRUNDLÄGGANDE REGLER

▼
1

Medlemskap i SJTK krävs för alla arrangemang om annat inte anges i inbjudan.

2

För icke-tävlingar kan arrangören göra undantag från medlemskravet, detta skall ske sparsamt per eventuell ny medlem.

3

Vid Open-tävling behövs inte medlemskap i SJTK. Par i dubbelklass kan bestå av medlem och icke-medlem. Anmälningsavgiften gäller per person.

4

Spelform: KM- och OPEN-spelform: HSA i "Bäst av tre set med tiebreak." Övriga klasser i "Bäst av tre set med NoAd och tiebreak i de två första seten samt supertiebreak i avgörande set."

🚨 Vid överanmälan kan följande åtgärder vidtas:

  • a) Tävlingsledningen kan korta spelformen i alla klasser (HSA)
  • b) "Icke-seedad spelare/par garanterad två matcher" kan återkallas
  • c) Möjlighet att äska mer tid i STC-hallen (kontakt via STC-shop minst 3 dagar innan)
⚖️

KM- och OPEN-tävling: B-klass ska ej fyllas upp för att uppnå ett "jämnt startfält".

👥

Om arrangemang för "Herrar" utlyses, så skall även motsvarande arrangemang för "Damer" utlysas.

💰

Presumtiv deltagare måste vara skuldfri till SJTK för att anmälan skall godtas.

🎲 LOTTNINGSFÖRFARANDE

▼
❌

Färre än 3 deltagare

Arrangemanget/tävlingen ställs in. Inbetald anmälningsavgift återbetalas.

🔄

3-4 deltagare

Pool-lottning där alla spelare/lag möter alla de övriga spelare/lag.

📊

Fler än 3 deltagare

Stegvis reducerande schema med radtal: 4, 8, 16, 32 osv.

Om antalet deltagare ligger mellan dessa tal används närmast större lottningsschema.

💡 Lottning och fördelning följer principer från tävlingshandboken för seedade spelare/lag.

🏆 SEEDNING

▼

📋 Bedömningsprocess

1

Tävlingskommittén bedömer spelstyrkor

2

Presenterar rekommendation

3

Tävlingsledaren fattar slutgiltigt beslut

📊 Bedömningskriterier

  • 🏅 Klubbrankning
  • 📈 Styrkt resultatförteckning
  • 👤 Personlig kännedom om kapacitet
  • 🏠 Kontakt med hemmaklubbar (utomstående)

📏 Antal seedade per klasstorlek

Antal deltagare Antal seedade
4-8 2/4
9-49 4/8
49+ 16/32

⏰ EFTERANMÄLDA SPELARE OCH VAKANSER

▼

📅 Efteranmälan

Spelare/lag som anmäler sig efter anmälningstidens utgång men före första matchstart får delta om tävlingsledningen så bestämmer.

🚫 Uteblivna spelare

Om spelare/lag inte infinner sig till matchstart får tävlingsledaren besluta att utdöma (W.O).

🎾 BOLLAR

▼

🏆 Tävlingskommitténs ansvar

Beslutar om bollmärke med minst 4 bollar per förpackning.

👨‍💼 Tävlingsledarens ansvar

Avgör fördelningen av nya respektive använda bollförpackningar mellan klasser och matcher.

⚖️ SPEL UTAN DOMARE

▼

🏛️ Grundprinciper

👁️

Spelarna ansvarar för domslut på sin sida av nätet

📢

Alla "ute" och "fel" domslut måste göras omedelbart och hörbart

❓

Vid osäkerhet ska bollen dömas till motståndarens fördel

🔄

Felaktiga domslut som korrigeras ger poäng till motståndaren

🎾 Serveregler

Servaren ska meddela poängställningen före varje poängs början, så högt att motståndaren kan höra det.

🟫 Särskilt för grusbana

  • Bollmärke kan kollas för slag som avslutat poängen
  • Spelare kan be motståndaren visa märket vid osäkerhet
  • Radering av märke = poäng till motståndaren

👨‍💼 Tillkalla tävlingsledare

✅ Endast om icke-spelande tävlingsledare finns på plats

❌ Får ej störa spelande tävlingsledare

🤝 SJTK är en trivselklubb - lös problem tillsammans

⚠️ Viktiga påminnelser

  • Spelarna ska följa tävlingsledarens anvisningar
  • Osporsligt beteende kan leda till uteslutning
  • Tävlingsledaren avgör frågor om "spel utan domare"
